# astro_remote
Astro Byond Decoder / STB Network Remote Controller

## Turn On Decoder
```
  java Main state on
```

## Turn off decoder
```
  java Main state off
```

## Change channel to 834
```
  java Main channel 834
```
  
## Change channel to 532
```
  java Main channel 532
```
  
## Previous channel
```
  java Main previous
```
  
## Next channel
```
  java Main next
```
